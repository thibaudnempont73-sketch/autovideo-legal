Politique de confidentialite

Derniere mise a jour : avril 2026

1. Nature de l'application

AutoVideo est une application de bureau (desktop) qui s'execute entierement en local sur l'ordinateur de l'utilisateur. Aucune donnee n'est transmise a nos serveurs : nous n'en avons pas.

2. Donnees collectees et stockage

Toutes les donnees sont stockees localement sur la machine de l'utilisateur, dans une base de donnees SQLite et des dossiers locaux :

  • Tokens OAuth (access_token, refresh_token) pour les plateformes connectees
  • Metadonnees des videos generees (titre, description, tags, script)
  • Fichiers video et audio temporaires
  • Parametres de configuration (cles API, preferences)

3. Transmission aux tiers

L'application communique avec les services tiers suivants uniquement lorsque l'utilisateur declenche une action :

  • Anthropic Claude API : generation de scripts textuels
  • Pexels API : recherche et telechargement de clips video stock
  • Microsoft Edge TTS / gTTS : synthese vocale
  • TikTok Content Posting API : upload et publication de videos

Aucune donnee personnelle n'est vendue, partagee ou transmise a d'autres tiers.

4. Utilisation de l'API TikTok

L'application utilise l'API officielle TikTok Content Posting. Les donnees transmises a TikTok sont strictement limitees a :

  • Le fichier video a publier
  • Les metadonnees de publication (titre, description, hashtags)
  • Les parametres de confidentialite (prive / public / amis)
  • Le marquage AIGC (contenu genere par IA)

Les scopes OAuth demandes sont : user.info.basic (nom d'utilisateur uniquement), video.publish (publication directe), video.upload (upload en brouillon).

5. Securite des donnees

Les tokens OAuth sont stockes dans la base de donnees SQLite locale de l'utilisateur. L'utilisateur est responsable de la securite de sa machine et de l'acces a cette base.

6. Suppression et controle

L'utilisateur peut a tout moment :

  • Supprimer un compte connecte depuis l'application (efface le token localement)
  • Revoquer l'autorisation OAuth depuis les parametres de TikTok/Instagram/YouTube (invalide le token cote plateforme)
  • Supprimer toute l'application et ses donnees en effacant le dossier d'installation

7. Enfants

AutoVideo n'est pas destinee aux personnes de moins de 13 ans. L'utilisateur doit se conformer aux conditions d'age des plateformes cibles (TikTok : 13+).

8. Contact

Pour toute question relative a la confidentialite ou aux donnees, contactez-nous via l'application.
